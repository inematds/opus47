🎬 O que é este curso

Este é um curso prático, ilustrado e denso construído a partir das três referências oficiais da Anthropic publicadas em abril de 2026: Best Practices para Opus 4.7, Session Management e 1M context, e o guia de Prompting Best Practices.

Ao final das 4 trilhas, você:

  • ✓Extrai 2–3× mais valor por hora do 4.7 vs. seu baseline do 4.6
  • ✓Escreve briefings ICCA que eliminam drip-feed
  • ✓Orquestra subagentes e auto mode com segurança

🛠️ Setup em 5 minutos

💡 Dica Prática

Se você já usa Claude Code e nunca setou effort manualmente, já está no 4.7 com xhigh. Rode /usage para confirmar.

1. Atualize o Claude Code

claude --upgrade

2. Confirme modelo e effort

/usage
# procure por: model: claude-opus-4-7 | effort: xhigh

3. (Opcional) Configure effort alternativo

# Na próxima sessão, use /effort para trocar
/effort high   # ou xhigh, max, medium, low
